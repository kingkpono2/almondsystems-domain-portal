AlmondShipping adds market-ready Nigerian delivery rates to WooCommerce, including Mile 2 based Lagos pricing, Nigerian state rates, checkout autosuggest, and admin rate management.

== Description ==

AlmondShipping is a WooCommerce shipping method for merchants who deliver across Lagos and Nigeria. It includes seeded Nigerian shipping rates, a Mile 2 central Lagos base point, editable admin rate tables, and a customer-friendly checkout city/area autosuggest field.

Features:
* WooCommerce shipping-zone compatible method.
* Seeded Lagos city/area rates using Mile 2 as the base point.
* Seeded rates for Nigerian states using WooCommerce state codes.
* Checkout label configurable from WooCommerce shipping settings.
* Type-ahead datalist field for delivery city/area.
* Admin page under WooCommerce > AlmondShipping.
* Activation migration and rate seeding.
* Legacy BuySmartNG method compatibility for stores moving from the theme module.

== Installation ==

1. Upload the `almondshipping` folder to `/wp-content/plugins/`.
2. Activate the plugin from WordPress Plugins.
3. Go to WooCommerce > Settings > Shipping and confirm the Nigeria shipping zone has AlmondShipping enabled.
4. Go to WooCommerce > AlmondShipping to edit rates.

== Frequently Asked Questions ==

= Does it expose any third-party source to customers? =
No. The checkout method label is controlled by the store owner and defaults to AlmondShipping.

= Can I change the rates? =
Yes. Go to WooCommerce > AlmondShipping and edit the Lagos and Nigerian state rates.

= What is the checkout city/area field? =
It is a type-ahead datalist input. Customers can type related Nigerian cities or Lagos areas and choose from suggestions.
